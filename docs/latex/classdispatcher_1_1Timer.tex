\subsubsection*{template$<$typename Network$>$\newline
class dispatcher\+::\+Timer$<$ Network $>$}

A timer utility for scheduling tasks in the event loop. 

The {\ttfamily \mbox{\hyperlink{classdispatcher_1_1Timer}{Timer}}} class provides functionality to schedule tasks to be executed after a specified duration or repeatedly at regular intervals. It uses the event loop associated with the specified network to handle the execution of tasks asynchronously.
